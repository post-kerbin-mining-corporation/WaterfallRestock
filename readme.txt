=================================
Waterfall - Restock Configs 0.2.0
=================================

This is a configuration pack for the Waterfall mod, which applies Waterfall type effects to Restock and Restock+ models. It does nothing on its own and requires the base Waterfall mod to function.

============
DEPENDENCIES
============

Required:
- Waterfall (0.5.0)
- Restock (1.3.1+)
- ModuleManager (v4.1.4)

============
INSTALLATION
============

To install, place the GameData folder inside your Kerbal Space Program folder. If asked to overwrite files, please do so.

=======
KSP-AVC
=======

This mod is compatible with KSP-AVC version checking. For a comprehensive version checking experience, please download the appropriate plugin (http://forum.kerbalspaceprogram.com/threads/79745).
